## Projects

### 🔵 SOC Automation Pipeline
**Stack:** Wazuh · Splunk · Sysmon · n8n SOAR · TheHive

A fully functional SOC home lab simulating real-world alert triage and automated incident response. Wazuh ingests endpoint telemetry via Sysmon, forwards enriched alerts to Splunk for correlation, and triggers n8n SOAR playbooks to auto-create and escalate cases in TheHive — reducing manual analyst effort from detection to case creation.

**Key outcomes:**
- End-to-end alert pipeline from endpoint event → SIEM → SOAR → case management
- Custom n8n playbooks for automated IOC enrichment and case assignment
- Demonstrates Tier 1 SOC workflows: log analysis, triage, escalation
